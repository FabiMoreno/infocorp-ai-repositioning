Qué hace

Cuatro fundamentos
de una web que convierte

No alcanza con que la web cargue rápido. Tiene que ser igual de buena en todas las pantallas y conectarse con el resto del banco sin esfuerzo.

Touch-first design

Áreas táctiles de 44px+, gestos nativos, formularios que respetan el teclado y la zoom-safety. Pensada para el dedo, escala al mouse.

Design system compartido

Mismo lenguaje visual y de interacción que IC Mobile Banking. El cliente abre web o app y sabe instintivamente dónde está.

Performance budget

Lighthouse 96+, LCP < 1.8s, CLS < 0.05. Auditado en cada deploy. Si una feature rompe el budget, vuelve a refactor.

Accesibilidad WCAG 2.1 AA

Contrast ratios, navegación por teclado, screen reader compatible. Compliance regulatorio + audiencia que crece todos los días.
